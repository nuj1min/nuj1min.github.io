About

코드 너머, '사람'과 '시스템'을 잇는 개발자

"개발의 본질은 결국 '소통'과 '문제 해결'에 있으며
이것이 팀과 기술을 연결한다고 믿고 있습니다."

서버 인프라를 구축하고 트래픽을 제어하는 기술적 역량만큼이나, 제가 가장 중요하게 생각하는 것은 투명하고 유기적인 소통입니다. 양질의 결과물은 뛰어난 개인의 코드가 아닌, 한 몸처럼 움직이는 팀의 단단한 결속력에서 비로소 완성된다고 믿기 때문입니다. 학과 학생회부터 보안 학회, GDG, UMC 등 다양한 조직에서 기획 및 리드 역할을 수행하며 대규모 인원의 의견을 조율하고, 팀원들이 하나의 목표를 향해 나아가도록 이끄는 협업의 가치를 깊게 체득해 왔습니다.

이러한 소통 능력의 기반에는 새로운 기술과 낯선 과제 앞에서도 절대 물러서지 않는 자신감과 열정이 있습니다. 프로젝트 중 한 번도 다뤄보지 않은 기술 스택이나 복잡한 요구사항이 주어지더라도 결코 두렵지 않습니다. 필요하다면 며칠 밤을 지새워서라도 기어코 원리를 파악하고 묵묵히 구현해 내고야 맙니다.

학업과 교내 행정 실무를 병행하며, 매일 운동을 통해 빚어낸 단단한 체력과 끈기는 제가 수많은 협업을 성공적으로 완주할 수 있었던 가장 강력한 원동력입니다. 팀원들과 끊임없이 교감하며 최적의 아키텍처를 설계하고, 풀리지 않는 에러 앞에서도 결코 타협하지 않고 돌파해 내는 든든한 백엔드/클라우드 엔지니어 김민준입니다.

whatshu : GDG 출석체크 서비스
프로젝트 개요

GDG(Google Developer Groups) 캠퍼스 활동 중, 기존 구글폼 기반의 번거로운 행사 출석 인증 방식을 혁신하기 위해 기획된 모바일 웹 출결 관리 서비스입니다. 수기 관리의 휴먼 에러를 줄이고 운영진과 멤버 모두에게 쾌적한 행사 경험을 제공하는 것을 목표로 했습니다.

핵심 기술적 성과 및 역할
  • 인증 아키텍처 리딩 (JWT & Security): 전통적인 세션 방식 대신 JWT(JSON Web Token)를 도입하여 서버의 무상태성(Stateless)을 유지하고 확장성 있는 인증 구조를 설계했습니다. Spring Security와 연동하여 안정적인 인가 프로세스를 구축했습니다.
  • 배포 자동화 인프라 구축 (Docker & AWS): Docker 컨테이너 기술을 활용해 개발-운영 환경을 일치시켰으며, AWS 인프라 위에 CI/CD 파이프라인을 구축했습니다. 이를 통해 팀의 배포 속도를 높이고 인프라 관리 리소스를 획기적으로 줄였습니다.
  • 백엔드 시스템 설계: Java / Spring Boot / MySQL을 활용하여 고부하 상황에서도 안정적인 출결 데이터 처리가 가능하도록 데이터베이스 스키마와 API를 설계했습니다.
프로젝트 임팩트

도입 결과, 기존 평균 5분 이상 소요되던 출석 체크 시간이 10초 이내로 단축되었으며 운영진의 데이터 정리 업무 시간을 80% 이상 절감하는 성과를 거두었습니다.

외대판 길건너 친구들
프로젝트 개요

인기 아케이드 게임 '길건너 친구들'의 메커니즘을 한국외대 글로벌캠퍼스의 지형과 건물로 재해석한 3D 아케이드 게임 프로젝트입니다. 단순한 모작을 넘어, Unity 엔진의 핵심 사이클을 깊이 있게 이해하고 팀 프로젝트의 전체 생명주기를 관리하는 데 집중했습니다.

기술적 성과 및 리더십
  • 팀 리딩 및 PM (Project Management): 프로젝트 팀장으로서 전체 일정 관리와 기능 명세(Specs)를 정의했습니다. 팀원 각자의 강점에 맞게 업무(에셋 제작, 물리 로직, UI/UX)를 분배하고 코드 통합 시 발생하는 충돌을 조율했습니다.
  • 객체 지향 설계 및 물리 시스템 구현 (C# & Unity): 캐릭터의 정교한 이동 시스템, 동적인 장애물(차량/기차) 스폰 알고리즘, 그리고 레이캐스트(Raycast)를 활용한 충돌 판정 로직을 직접 구현하며 OOP 설계 역량을 강화했습니다.
  • 사용자 경험(UX) 최적화: 게임의 긴장감을 더하는 점수 시스템, 캠퍼스 맵의 특징을 살린 이벤트 트리거, 그리고 직관적인 게임 종료 UI를 통해 완성도 높은 플레이 경험을 제공했습니다.
